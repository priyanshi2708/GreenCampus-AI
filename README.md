# 🌱 GreenCampus AI

An intelligent, gamified SaaS platform that helps campuses track, predict, and optimize resource consumption and carbon footprint metrics using AI-driven insights.

## 🌟 Overview

GreenCampus AI provides smart sustainability monitoring by analyzing campus resources such as:

* ⚡ Energy consumption
* 💧 Water usage
* ♻️ Waste management
* 🌍 Carbon footprint

The platform uses AI-based prediction and analytics to generate insights, alerts, and recommendations for creating greener campuses.

---

## 🚀 Features

### 📊 Smart Dashboard

* Real-time sustainability metrics
* Resource consumption visualization
* Carbon footprint tracking

### 🤖 AI Prediction System

* Predicts future resource usage
* Identifies consumption patterns
* Provides optimization suggestions

### 🔔 Smart Alerts

* Detects abnormal usage
* Sends sustainability warnings

### 🎮 Gamification

* Sustainability score
* Green challenges
* Campus ranking system

## 💻 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/GreenCampus-AI.git
```

Go to project folder:

```bash
cd GreenCampus-AI
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
